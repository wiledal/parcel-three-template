# Parcel THREE.js template
Basic starter template for fast THREE.js "demo"-style project development with `Parcel`.

*Features:*
- Scene, Camera, Renderer
- Window resizing
- Update loop

## Get
```bash
npx degit wiledal/parcel-three-template PROJECT_NAME
```

## Install
```bash
npm install
```

## Run
```bash
npm run dev
```
